Self-Driving RC Car

An RC car built with an Arduino Uno board that can autonomously follow lines of black tape.

Self-Driving RC Car
Previous Next

This RC Car was built for a project in my Computer Engineering class in highschool. Using an Arduino Uno board, my group was able to develop an RC car that could follow a track of black tape without the need for human input while it is running. This was achieved by installing two colour sensors on the vehicle. We programmed the colour sensors (using embedded C) to turn the wheels accordingly whenever it detected black tape.

GoFindMe

Lost something? Use my app to make your community find it for you.

GoFindMe
Previous Next

This app helps users find things that they have lost. Using Estimote Beacons, people will be able to detect lost items while outside. You could find someone's lost bike while taking a morning walk through the park with the help of this app. If you find the lost object, you are given the owner's contact info to get in touch with them. A reward may be in store for the person who helps the user find their lost item.
In the future our team would like to implement an integration with Google Maps so users can visualize the position of lost items. A secure payment system is another feature that we would like to add in the future. This would allow users to safely reward someone who found their lost item without the need to exchange money in real life. My team learned a lot about how to use a third-party API to communicate with bluetooth devices. I learned more about the Android Studio framework, as well as Java.
Go to Github Repo
